	Usage: SYS 

	If you give NO argument yaze-ag goes in an interactive modus of the
	monitor.

	If you give "help " you will see detail informations of the
	command.

	The monitor commands are described in yaze.doc and in yaze-ag.doc and 
	in the manual page yaze(1).
