YAZE-AG     

    OS          


	Yaze-ag-2.51.0 runs on the following operating systems:

		Unix/Solaris (e.g. Sparcv9)
		Linux	(also on all Raspberry Pi's)
		FreeBSD
		MacOS-X
		cygwin (Windows) (since version 2.01)

	If you install the source package you have only to edit the
	Makefile to set the processor type.

	PLEASE read INSTALL-2.51.0 for informations about the install process!

	For Windows there gives a binary versions on the web page.
